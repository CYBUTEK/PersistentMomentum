Description

This mod allows for momentum to persist through time warp and game saves/loads. Allowing the creation of rotational stabilised satellites, centripetal space stations and an all round more realistic docking experience. It also adds difficulty via not being able to rely on freezing your momentum by quickly enabling and disabling time warp.

Installation

  • Copy the 'PersistentMomentum' folder into the 'GameData' folder located within your Kerbal Space Program installation directory.

Notes of Interest

  • The part module used for persisting momentum will be installed dynamically onto all parts (including mods) when starting Kerbal Space Program.
  • The part module used for persisting momentum will not be installed retractively onto parts that are already in flight. Only newly launched vessels.
